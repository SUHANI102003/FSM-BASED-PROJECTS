### ABOUT FSM DESIGN

<p>Design an FSM with 1 Mealy output y. The output should be asserted when either of the following sequences is detected:</p>
<ul>
  <li>010</li>
   <li>1001</li>
</ul>
<p>Overlapping is allowed. No reset necessary</p>
